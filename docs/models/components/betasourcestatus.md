# BetaSourceStatus

To control which sources to be shown regarding their beta status


## Values

| Name                          | Value                         |
| ----------------------------- | ----------------------------- |
| `BetaSourceStatusLiveAndBeta` | LiveAndBeta                   |
| `BetaSourceStatusOnlyLive`    | OnlyLive                      |
